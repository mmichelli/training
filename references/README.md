## How to refresh

Re-fetch source URLs from `../principles.md` (#References) and update the markdown here. If a paper's conclusions change in a later version, the citation in `principles.md` must change too — and the audit note there must be updated.

## Why we keep these locally

1. Links rot. Open-access PMC URLs and journal URLs have changed hosts before.
2. Citations made up by LLMs are easy to debunk if you keep the actual papers next to the claims they support.
3. The coach + the dashboard ground in these — if the evidence base is local and inspectable, anyone reading the project can check the work without leaving the repo.
